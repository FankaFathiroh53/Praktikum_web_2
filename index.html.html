Soal 1: Mengapa menggunakan CSS dan JavaScript dalam pemrograman web?

Penggunaan CSS (Cassading Style Sheets) dalam pemograman web untuk mendesain tampilan halaman website berupa tampilan visualnya (warna, font, layout, dan space) agar terlihat menarik dan responsif yang dapat menyesuaikan desain sesuai ukuran layar perangkat pengguna (desktop, tablet, atau smartphone). Penggunaan JavaScript merupakan suatu bahasa pemograman yang digunakan untuk menambahkan halaman website agar interaktivitas (munculnya tombol, tampil animasi dan pembuatan formulir) dan menjaga pola dinamis halaman untuk menambah, menghapus, dan mengubah konten halaman tanpa perlu refresh laman web.

Soal 2: Bagaimana CSS dan JavaScript dapat diintegrasikan dalam pemrograman web?

Pengintegrasian antara CSS dan JavaScript akan menghasilkan website yang menarik secara visual dan interaktif. Dalam penggunaannya JavaScript dapat langsung digunakan dalam CSS untuk mengatur warna halaman, mengontrol tampilan agar dapat berinteraksi dengan pengguna website. Selain itu penulisan CSS dalam JavaScript dapat menjadikan website yang dinamis, responsif dan realtime.

Soal 3: Jelaskan apa saja keuntungan menggunakan JavaScipt dan CSS?

Keuntungan CSS adalah menampilkan kesan visual laman web dan desain yang responsif. Tampilan website yang dibuat juga dapat menyesuaikan berbagai ukuran layar. CSS juga mendukung penggunaan class dan IDE sehingga mengurangi duplikasi kode dan mempercepat proses pengembangan website. Pada JavaScript juga memberikan keuntungan dalam membangun sebuah website. Website yang dibuat terkesan realtime. Dalam JavaScript memungkinkan manipulasi elemen HTML secara langsung melalui DOM(Document Object Model).

Soal 4: Jelaskan apa saja keuntungan menggunakan Bootstrap?

Keuntungan utama dalam menggunakan Bootstrap adalah penggunaan sistem grid fleksibel sehingga pengembang mudah dalam membuat layout secara responsif. Bootstrap juga banyak menyediakan komponen yang siap dipakai tanpa desain dahulu diawal, sehingga menghemat waktu untuk menulis kode CSS, dan juga penggunaan bootstrap menyediakan desain default sehingga layar yang ditampilkan menyesuaikan pada kebutuhan. Bootstrap memungkinkan penggunaan kode CSS untuk mengubah warna tema, font, dan spacing.
